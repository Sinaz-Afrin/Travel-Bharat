# 🇮🇳 The Travel Bharat – Soul of India

A modern and responsive **India Tourism Web Application** built to help users discover the beauty, culture, heritage, and destinations across India.  
This project showcases famous tourist places state-wise, region-wise, and category-wise with a smooth user experience and attractive UI.

> Designed and developed as a personal portfolio project to demonstrate frontend development, UI/UX design, routing, and scalable project architecture.

# ✨ Features

## 🏠 Home Page

* Beautiful hero section
* Featured destinations
* Explore by categories
* Popular states and regions
* Smooth responsive layout

## 🗺️ State Tourism Pages

Dedicated pages for each state and union territory including:

* Hero banner with state name
* State history / specialty overview
* Top attractions
* Best time to visit
* Travel tips
* Image gallery
* Responsive layouts

## 📍 Place Details Pages

Each tourist destination includes:

* Full description
* Image gallery slider
* Nearby attractions
* Quick info section
* Back navigation to state page

## 🔍 Smart Search

Search destinations by:

* Place name
* State
* Category

## 🌍 All Destinations Page

Browse all places with filters:

* Region
* State
* Category

## 📱 Fully Responsive

Optimized for:

* Desktop
* Tablet
* Mobile

# 🚀 Installation & Setup

Clone the repository:
```
git clone https://github.com/yourusername/travel-bharat.git
```

Move into project folder:
```
cd travel-bharat
```

Install dependencies:
```
npm install
```

Start development server:
```frontend
npm run dev
```

```backend
cd backend
npm run dev
```
